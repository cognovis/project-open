Monitoring DB para o Postresql

Monitora o tamanho do banco de dados e da pasta content-repository!

PostgreSQL 7.4 até 8.0

Precisa compilar o aplicativo dbsize que fica na pasta contrib!

#cd /usr/local/src/postgresql-7.4.9/contrib/dbsize
#make
#make install

aplicar o dbsize.sql que ele a instalação do dbsize gera na pasta dbsize para o banco de dados que está usando!

# psql banco_de_dados < /usr/local/src/postgresql-7.4.9/contrib/dbsize/dbsize.sql

PostgreSQL 8.1 ou superior

É preciso setar o parametro VersionDB do monitoring como "8.1"
