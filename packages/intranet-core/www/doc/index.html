]project-open[ Core

P/O Core defines the base functionality of the ]project-open[ product family. P/O is build around the idea of extensibility, allowing for application modules to extend the functionality of other modules. This extensibility is implemented using a component / glue-code philosopy, using HTML widgets (TCL procedures) as "components" and using TCL code stored in the database as glue-code.
